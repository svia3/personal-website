ECE 3056: Architecture, Concurrency, and Energy

Cache and VM Simulator

This class was all about memory management and cpu modeling. It really helped me develop an appreciation for all of the work that compilers, registers, and datapaths do in the processes that make up computing power. This project had two main goals:

  • simulate a single level, N-way set associative cache for a given block size, using LRU replacement
  • simulate the paging system of an OS Kernel by performing a single process with a page table and TLB

VHDL MIPS Pipeline

VHDL stands for "VHSIC Hardware Description Language." VHSIC is yet another achronym which stands for "Very High Speed Integrated Circuits." It is important because it can model the hardware interactions of a processor. MIPS is a 32-bit, reduced-instruction set architecture that many processors nowadays utilize. Basically, this project was aimed at students attempting to make their own pipeline model in VHDL that implemented data-forwarding, and supported load-to-use stalls and control hazards.
